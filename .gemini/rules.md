# Abelink Rules & Constraints

## Supabase MCP Policy
- **RESTRICTED**: Supabase MCP tool (write tables/DDL) is ONLY authorized for:
  - `LearnInk`
  - `AbPay`
- **PROCEDURE**: For `Abelink-Portofolio` or other projects, do NOT use MCP to execute DDL/SQL. Instead, provide the SQL script to the user for manual execution in the Supabase Dashboard.
